🚀 产品与功能更新

**本板块主线：**AI产品正从通用聊天界面，向深度的垂直领域工作流（科研/医药）与物理世界的具身化（医疗硬件/外骨骼）双重突进。

Anthropic推出Claude Science：大模型竞争从“通用问答”转向“垂直工作流接管”

核心判断（证据强度：强）：大模型的竞争焦点正从“通用能力跑分”转向“垂直领域工作流整合”，Claude Science标志着AI开始实质性地重构高价值科研与药物研发的工具链。

  • 推演链条：Anthropic发布Claude Science，并非简单推出一个新模型，而是打造了一个集成碎片化工具和数据集的“工作台”。这表明Anthropic意识到，在通用模型能力差异逐渐缩小的阶段，谁能把AI无缝嵌入科学家的高价值工作流（如数据分析、图表生成、药物筛选），谁就能锁定最优质的B端付费场景。结合OpenAI、Anthropic和Google CEO联合呼吁制定DNA筛查法律（Google News），可以看出头部AI厂商正在为AI深度介入生物医疗和基因领域提前铺设合规与伦理基础设施。Google Workspace用AI重构创作流程的广告（TechCrunch）也是同一逻辑：AI正在成为各行业底层的工作操作系统。
  • 实战建议：对于创业公司而言，做“套壳ChatGPT”的窗口已彻底关闭，真正的机会在于像Anthropic一样，深入某个垂直行业（如法律、医疗、材料），将大模型能力转化为该行业专用的“工作台”产品，绑定工作流而非仅提供对话框。

Midjourney跨界医疗硬件与傲鲨外骨骼落地：AI产品向物理世界“具身化”突进

核心判断（证据强度：中）：AI公司不再满足于纯数字维度的输出，开始试图通过自研硬件（如Midjourney医疗扫描仪）或赋能传统硬件（如傲鲨外骨骼）将能力落地到物理世界，但硬件的信任与验证壁垒远高于软件。

  • 推演链条：Midjourney以AI图像生成闻名，却展示了其未来的医疗超声扫描仪，但至今未提供其有效性的实质证明（The Verge）；与此同时，傲鲨将外骨骼机器人从工业场景推向消费级户外徒步，通过AI学习行走姿态提供推力（36kr）。这两者看似无关，实则指向同一趋势：纯软件的生成式AI正面临天花板（或版权争议，这正是Midjourney要求好莱坞披露AI使用细节的原因TechCrunch），AI必须寻找新的物理载体。Midjourney选择“降维打击”进入医疗成像，傲鲨选择“升维赋能”让机甲消费化，都是AI试图接管物理世界信号的体现。Dylan Patel预测OpenAI和Anthropic到2030年需要超100GW算力（X），这种推理算力的狂飙，很大程度上正是为了支撑具身智能和物理世界模拟。
  • 不确定性：Midjourney的医疗扫描仪目前更像是概念验证，缺乏临床数据和同行评审，从“能生成图像”到“能精准扫描并分割物理组织”存在巨大的工程与医学鸿沟，存在严重翻车风险；傲鲨外骨骼的AI自适应能力在复杂户外路况下的可靠性也有待更大规模验证。
  • 反向视角：Midjourney跨界医疗硬件的步子迈得极大，这不仅是技术挑战，更是监管挑战。在FDA等严苛监管下，软件迭代“快速试错”的逻辑在医疗硬件上完全行不通，这可能最终被证明是一场营销大于实质的秀。

🔬 前沿研究

**本板块主线：**AI代码Agent的自主化引发了安全攻防与能力对齐的剧烈冲撞，同时推理架构正打破自回归瓶颈向并行扩散演进，以适配Agent的高吞吐需求。

AI代码Agent引发安全攻防失衡，前沿模型付出高昂“对齐税”

核心判断（证据强度：强）：AI在代码与漏洞挖掘领域的Agent化部署，已引发高严重性CVE的爆炸式增长，迫使前沿模型在安全与能力间付出实质性的“对齐税”。

  • 推演链条：Anthropic重新发布Claude Fable 5，因新增网络安全分类器拦截网安任务，导致其在APEX-SWE基准上的Pass@1从6月的65.5%骤降至54.8%（尽管仍优于Opus 4.8的45.3%）AnthropicAI mercor_ai。这种能力折损并非孤例，而是宏观趋势的缩影：2026年6月高严重性CVE报告数量激增至先前月记录的3.5倍以上，直接归因于AI驱动的漏洞挖掘程序上线the-decoder；Mistral的Leanstral 1.5也在57个开源仓库中揪出了5个真实漏洞the-decoder。同时，arXiv新研究揭示了持久化状态下的编码Agent极易遭受跨PR的分布式投毒攻击arXiv。这形成了一个闭环：Agent代码能力越强 -> 安全风险与攻击面指数级放大 -> 模型被迫加严安全护栏 -> 通用代码与调试能力受损。
  • 实战建议：单纯依靠输出端分类器拦截的“外挂式安全”已触及能力天花板，未来防御需向底层转移，如采用LACUNA这类针对特定参数的精准遗忘方案arXiv，或部署基于校准风险控制的在线实时监控器arXiv，以降低对正常编码任务的误杀率。

自回归推理架构现破局点，基准评测严重滞后于Agent真实算力上限

核心判断（证据强度：强）：NVIDIA双塔扩散架构打破了逐Token生成的延迟瓶颈，且当前基准评测正暴露出对Agent算力弹性的严重低估。

  • 推演链条：NVIDIA发布Nemotron-Labs-TwoTower，将30B模型拆分为“上下文记忆塔”和“Token写入塔”，通过扩散语言模型实现并行Token生成，无需从头重训即可大幅提升推理速度NVIDIAAI LiorOnAI。这不仅是架构创新，更呼应了NVIDIA将AI从“模型训练”转向“always-on token production”的商业叙事nvidia。然而，评测体系并未跟上这一架构与Agent范式的跃迁：英国AI安全研究所证实，在7大基准测试中，标准评测因人为限制算力预算，系统性地低估了Agent能力——在SWE任务中，仅将Token预算增加10倍，成功率就跃升约25%the-decoder。当前静态、算力紧缩的基准测试正在失效，无法反映Agent在真实高算力环境下的威胁水平与任务表现。
  • 反向视角：Token预算的无限放开在评测中提升了Agent成功率，但在工程实际中会遭遇延迟与成本墙，TwoTower这类并行架构正是为了缓解这一冲突而生。未来评测需将“算力-时间-准确率”联合建模，而非单纯看Pass@1。

模糊逻辑与物理/生物世界的编程范式重构

核心判断（证据强度：中）：LLM正在从确定性代码生成转向“权重化编程”，并与世界模型结合向非结构化物理与生物系统渗透。

  • 推演链条：arXiv提出Program-as-Weights范式，将自然语言规范的模糊函数（如日志告警、意图排序）直接编译为模型权重，取代脆弱的API调用arXiv；同时，中国团队将LeCun的JEPA架构引入细胞内部，构建微观尺度的世界模型qbitai。两者共同指向一个趋势：AI的“编程对象”正从确定性逻辑代码，拓展为概率性的模糊意图和复杂的动态物理/生物系统。
  • 不确定性：Program-as-Weights与细胞JEPA模型均处于早期学术概念验证阶段，缺乏大规模工程化部署案例；“权重化编程”在局部可解释性与跨模型复用性上仍存在巨大疑问。

🌍 行业展望与社会影响

**本板块主线：**AI产业链在资本端狂飙突进（算力扩产与Infra抢跑IPO），与在应用端加速割裂（合规摩擦与本土替代）形成鲜明对冲。

监管与商业化：AI工具合规摩擦加剧，倒逼大厂本土替代闭环

核心判断（证据强度：强）：外部AI工具的合规与安全风险正成为大厂推进内部替代的直接催化剂，AI编程等生产力工具的本土化商业化窗口被强制打开。

推演链条：阿里因“后门风险”全面禁用Claude Code并推荐自家的Qode 来源，与V2EX用户反映的Claude老号因网络环境变动被封禁 来源，看似是企业安全与个人风控的两个切面，实则指向同一逻辑：海外大模型在华使用的合规成本与业务连续性风险正在急剧上升。对于大厂而言，代码资产是核心命脉，安全风险是绝对红线；对于个人及小B端开发者，项目数据沉没成本极高。这种双重脆弱性，使得“国产平替”不再只是性能对比，而是刚性合规与业务避险需求。阿里借安全之名行商业替代之实，标志着AI应用层的竞争，已从技术指标竞争演变为“安全主权”与生态隔离的竞争。

  • 受影响对象：
    • 公司：Anthropic（在华渗透受阻）、阿里（Qode及通义系）、其他国产代码大模型厂（百度、字节等）
    • 岗位：企业信息安全合规岗、AI工具采购决策链
    • 产业链：AI编程助手赛道、企业大模型私有化部署服务商

实战建议：对于AI应用层创业者，当前做海外大模型的套壳或增值服务面临极高的政策与封号双重雷区，应迅速转向基于国产开源模型（如Qwen）的私有化/合规化工作流重构，这是当前最确定的商业化切入口。

资本：AI Infra与底层算力进入“军备竞赛”式的资本抢跑期

核心判断（证据强度：中）：在商业化盈利尚未充分验证前，AI Infra和算力硬件商正试图通过极速IPO和重资产锁仓来抢占资本和产能高位，将产业泡沫风险向二级市场和长周期转移。

  • 受影响对象：
    • 公司：硅基流动及同类AI Infra初创（如月之暗面、MiniMax等潜在的IPO跟随者）、美光/海力士等存储巨头
    • 岗位：港股18C上市保荐与审计团队、HBM先进封装工程师
    • 产业链：HBM制造与上游设备链、AI推理/训练云服务市场

不确定性：硅基流动的IPO可能仅是试探性递交，当前港股流动性能否承接未商业化AI公司的估值存疑；此外，美光2028年出货的产能规划基于当前大模型算力需求线性外推，若AI杀手级应用迟迟未能落地导致算力需求周期性回落，这些长周期重资产投资将面临严峻的产能过剩与债务危机。

Agent 能力正全面“Skill 化”，Claude Code 成为事实标准运行时

核心判断（证据强度：强）： AI 编程 Agent 的竞争焦点已从“底层模型能力”转向“技能生态与运行时标准”，Claude Code 正凭借极强的 Skill 扩展性占据事实标准地位。

  • 推演链条： 今日 GitHub Trending 出现罕见的“Skill 密集爆发”现象：mattpocock/skills（15.6万星）、JuliusBrussee/caveman（8.4万星）、alirezarezvani/claude-skills（2万星）以及规范项目 agentskills/agentskills 均指向同一趋势。其中 caveman 通过极端的提示词压缩（类 Caveman 语法）削减 65% Token，证明 Skill 不仅是功能扩展，更是性能与成本优化的核心手段。更关键的信号是 openai/codex-plugin-cc 的上榜——OpenAI 官方推出插件让自家 Codex 调用竞品 Claude Code 执行任务，这反向印证了 Claude Code 在工程侧（尤其是多文件编辑与终端控制）已成为当前不可绕过的运行时环境。
  • 反向视角： Skill 生态的繁荣也意味着“Prompt 正在资产化”。当核心业务逻辑被写成 Skill 文件分发，企业需警惕核心逻辑的外流与同质化；同时，过度依赖 Skill 组装可能掩盖底层模型指令遵循能力的不足。

“物理级越狱”：AI生产力刚需与风控/网络阻断的极限拉扯

核心判断（证据强度：中）：国内高端开发者对Claude等顶级AI工具的依赖已达到“不可逆”程度，风控封锁与网络审查正迫使采用成本从“软件代理”向“物理肉身/设备出海”升级。

推演链条：

  1. 风控倒逼物理隔离：在终极折腾方案中，一名安全顾问尝试了VPS自建、静态住宅IP、虚拟信用卡（WildCard等）、设备环境伪装等几乎所有“软件级”绕行手段，均遭遇封号或阻断，最终被迫考虑“海外拿身份+境外友人放Mac mini”的物理隔离方案。这说明Anthropic等厂商的风控模型已能精准识别VPS IP段、虚拟卡特征及代理环境，传统的黑灰产绕行手段对专业开发者而言已失效，合规成本指数级上升。

本板块主线： AI Agent 向高风险实体领域渗透，倒逼安全层从模型内生走向独立工程化沙箱，同时长周期复杂工作流对现有评测与回放机制提出了降维打击。

自主Agent安全层：从模型对齐外溢为独立运行时Harness

核心判断（证据强度：中）：自主Agent的安全防护正从模型层的对齐微调，外移至独立的执行沙箱与权限控制层，第三方Agent安全Harness市场开始成型。

  • 推演链条： Orca 作为传统的云安全厂商，推出针对自主 AI Agent 的安全层，其底层逻辑在于：当 Agent 具备调用工具（如 API、数据库、代码执行）的能力时，仅仅依赖模型侧的“对齐”来防止越界是极其脆弱的。必须在外部构建确定性的 Harness，拦截和审计每一次工具调用。这标志着 Agent 安全范式从“劝导模型不做恶”转向“物理上阻止模型做恶”，沙箱隔离与细粒度权限控制（RBAC/ABAC for Agents）成为刚需基建。
  • 依据： Orca provides safety layer for autonomous AI agents - Let's Data Science
  • 不确定性： Orca 具体的技术实现细节尚未完全公开，其安全层是基于底层系统级的 eBPF/容器隔离，还是仅仅做 API 级的代理网关拦截，将直接决定其防护深度与工程实用性。

高风险垂直领域的Agent工程：药物发现倒逼全链路评测与回放重构

核心判断（证据强度：中）：Anthropic 亲自下场长周期药物发现，印证了在试错成本极高的垂直领域，现有的通用 Agent 工作流与评测体系已失效，必须自研端到端的领域专属 Harness。

  • 推演链条： 新闻中诺华 CEO 提及 AI 可将研发时间从 12 年压缩至 7-8 年，成功率从 8% 提升至 16%。这意味着 Agent 需要执行跨尺度、长周期的复杂工具调用（从文献检索、靶点发现、分子生成到湿实验排期）。现有的 CLI/IDE Agent 工作流根本无法支撑这种长达数月、依赖外部真实世界反馈的异步执行链。要在这种高风险场景中落地，Anthropic 必须构建极其严苛的 Harness 工程：包括支持极高延迟的工具调用协议、能够处理真实物理世界反馈的状态机、以及用于归因分析的完整执行回放机制（当某候选药物毒性超标时，需回放溯源是哪次工具调用/推理引入的缺陷）。
  • 依据： Anthropic launches its own drug discovery programs to tackle diseases Big Pharma considers unprofitable
  • 不确定性： Anthropic 可能仅将其模型作为底层推理引擎接入现有制药公司的自动化实验室管线，而非亲自下场构建整个实验室自动化的 Agent Harness 架构。长周期回放与状态管理的工程难度极大，目前尚缺乏公开的技术验证。
  • 实战建议： 对于试图进入垂直工业领域的 Agent 开发团队，不要迷信通用 Agent 框架的 Eval 基准。应立即将工程重心转移到构建“长异步调用链的状态持久化”与“失败节点的精准回放调试”能力上，这是跨越 Demo 走向生产的核心瓶颈。

  • 推演链条：光大证券研报明确指出，AI与HPC导致数据中心热密度飙升，液冷成为必选项 来源。在液冷方案中，冷板式虽然当前主流，但浸没式液冷才是解决极高功率密度的终极形态，而浸没式液冷对冷却液的绝缘性、不燃性要求极高，PFPE是目前最契合的材料。同时，研报提及国内企业已突破G5级电子级氢氟酸工艺并进入一线晶圆厂，这意味着国内头部氟化工企业已经具备了“超净高纯”的工艺底座，将这部分产能复用到纯度要求同样极高的PFPE生产上，具有天然的一体化成本与技术优势。从“芯片清洗”到“芯片散热”，底层工艺相通，但散热市场的增量空间远大于湿电子化学品。
